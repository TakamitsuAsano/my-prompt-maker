# 🚀 ビジネス0=>1アクションスライド生成アプリ

経営AI研修プロンプト集を活用し、ビジネス課題（0→1）を解決するための具体的なアクションプランやスライド構成案を、生成AI（NotebookLMなど）を使って作成するための補助ツールです。

## 🎯 アプリの概要

このアプリは、社内に蓄積された「プロンプト集（CSV）」を検索・加工し、GeminiやNotebookLMに入力するための最適な指示書（プロンプト）を生成します。

**主な特徴:**
* **API Key不要**: 外部の有料APIを使用せず、セキュアかつ低コストで運用可能です。
* **シンプル入力**: プロンプト内の `【 】` で囲まれた部分を自動検出し、入力フォームを生成します。
* **NotebookLM連携**: 生成されたプロンプトと、どの資料（議事録など）をNotebookLMにアップロードすべきかのガイドを表示します。

💻 使い方（社内ユーザー向け）
アプリにアクセス: 共有されたURLを開きます。

シーンを探す: 左側のサイドバーで「分野」を選ぶか、「キーワード検索」で困っている課題（例：「議事録」「壁打ち」）を入力します。

状況を入力: 画面左側のフォームに、会社名や具体的な悩みなどを入力します。

プロンプトをコピー: 画面右側に生成されたプロンプトをコピーします。

NotebookLMへ: Google NotebookLM を開き、推奨される資料（議事録やPDF）をアップロードした後、コピーしたプロンプトをチャット欄に貼り付けて実行してください。

🛠 開発・運用ガイド（管理者向け）
1. ローカルでの実行方法

手元のPCで動作確認をする場合の手順です。

前提: Pythonがインストールされていること。

Bash
# リポジトリのクローン（またはフォルダへ移動）
cd my-prompt-app

# 必要なライブラリのインストール
pip install -r requirements.txt

# アプリの起動
streamlit run app.py
2. データの追加・更新方法

新しいプロンプトを追加したい場合は、以下の手順で行います。

CSVの準備: 指定のフォーマット（下記参照）でCSVを作成します。

ファイルの配置: data/ フォルダの中にCSVファイルを置きます。

再起動: アプリを（ローカルなら）再起動、Web上ならページをリロードすると自動で読み込まれます。

CSVフォーマットの注意点:

文字コードは UTF-8 または Shift-JIS に対応しています。

必ず「想定シーン」と「プロンプト本文」という列名を含めてください。

ファイル名にある「 - 〇〇.csv」の「〇〇」部分が、アプリ上の「カテゴリ名」として認識されます。

3. デプロイ（Web公開）方法

Streamlit Community Cloud を使用する場合:

このフォルダ一式を GitHub にプッシュします。

Streamlit Community Cloud にログインします。

「New app」を選択し、このリポジトリを指定して「Deploy」をクリックします。
